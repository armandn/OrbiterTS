Class ServerEvent

ServerEvent is a simple data class used to pass information from an application's Server object to registered event-listeners when a server event occurs. The ServerEvent class also defines constants representing the available server events.

Static Readonly RESET_UPC_STATS_RESULT

RESET_UPC_STATS_RESULT: "RESET_UPC_STATS_RESULT" = "RESET_UPC_STATS_RESULT"

Dispatched when the server reports the result of a reset-UPC-stats attempt by the current client.

To determine the result of the attempt, use getStatus(), which has the following possible return values:

  • Status.SUCCESS
  • Status.ERROR
  • Status.PERMISSION_DENIED

Static Readonly TIME_SYNC

TIME_SYNC: "TIME_SYNC" = "TIME_SYNC"

Dispatched when the server sends the current server time in response to an earlier invocation of Server's syncTime() method by the current client. Listeners for ServerEvent.TIME_SYNC can retrieve the approximate time on the server via Server's getServerTime() method.

Static Readonly UPC_PROCESSED

UPC_PROCESSED: "UPC_PROCESSED" = "UPC_PROCESSED"

Dispatched when the current client is watching for "processed UPC messages" and the server finishes processing an inbound UPC message. To watch for processed UPC messages, use the Server class's watchForProcessedUPCs()

Static Readonly WATCH_FOR_PROCESSED_UPCS_RESULT

WATCH_FOR_PROCESSED_UPCS_RESULT: "WATCH_FOR_PROCESSED_UPCS_RESULT" = "WATCH_FOR_PROCESSED_UPCS_RESULT"

Dispatched when the server reports the result of a watch-for-processed-UPCs attempt by the current client.

To determine the result of the attempt, use getStatus(), which has the following possible return values:

  • Status.SUCCESS
  • Status.ERROR
  • Status.ALREADY_WATCHING
  • Status.PERMISSION_DENIED

Methods

getStatus

  • getStatus(): string | undefined

  • Returns the status of the operation to which this event pertains.

    The getStatus() method's return value is always one of the Status class's constants. For example, if the Server.WATCH_FOR_PROCESSED_UPCS_RESULT event occurs in response to a successful watch-for-processed-UPCs attempt, getStatus() will return the value of Status.SUCCESS.

    Returns string | undefined

getUPCProcessingRecord

  • getUPCProcessingRecord(): UPCProcessingRecord | null | undefined

  • Returns the UPCProcessingRecord object for a UPC that was processed by Union Server. Applies to the ServerEvent.UPC_PROCESSED event only.

    Returns UPCProcessingRecord | null | undefined
